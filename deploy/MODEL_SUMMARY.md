# Clair-3B Model Summary

Clair-3B is a highly capable 3-billion parameter language model designed for advanced conversational AI, coding assistance, and complex reasoning tasks.

## Key Capabilities

- **Enhanced Knowledge**: Significantly improved capabilities in coding and mathematics through specialized training
- **Instruction Following**: Advanced ability to follow complex instructions and generate structured outputs, especially in JSON format
- **Structured Data**: Excellent understanding and generation of tables, JSON, and other structured formats
- **Long Context**: Supports 4,096 token context windows for coherent, extended conversations
- **Multilingual**: Fluent in 29+ languages including English, French, Spanish, Portuguese, German, Italian, Russian, Japanese, Korean, Vietnamese, Thai, Arabic, and more
- **System Prompt Resilience**: Highly adaptable to diverse system prompts for improved role-play and customization

## Technical Specifications

- **Parameters**: 3 billion
- **Architecture**: Transformer-based language model
- **Context Window**: 4,096 tokens
- **Format**: GGUF (F16)
- **Size**: 5.75 GB

## Use Cases

- 💻 Code generation, debugging, and optimization
- 🧮 Mathematical problem solving and reasoning
- 💬 Natural conversation and dialogue
- 📝 Text generation and content creation
- 📊 Structured data processing and JSON generation
- 🌍 Multilingual applications and translation
- 🎯 Task-specific assistance with custom system prompts
